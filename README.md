# Gracious tree labeling

## Документация

В doc/csp.pdf

## Компилация

```bash
./build.sh
```

Използват се два важни макроса там:

1. **MAX_SECONDS** ограничава времето за търсене с alarm(2). Ако не се дефинира,
не се слага аларма.
2. **HAVE_PYTHON** използва machine learning. Ако не се дефинира, се използва
апроксимация.
